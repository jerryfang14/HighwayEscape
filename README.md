# HighwayEscape
A car game which the goal is to dodge oncoming traffic by moving lanes.

Project Overview:
Highway Escape is a Java Swing game where the player controls a car and must dodge oncoming traffic on a 3 lane road. This game features multiple difficulties, collision detection, keyboard and mouse controls, and score tracking.

This project demonstrates user interface, object oriented programming and file input/output using Java.

This program:
- displays a login and registration screen
- allows player car to move left and right using arrow keys and clicking the program itself
- spawns enemy cars that move downward toward the player
- has multiple difficulties that affect enemy spawn rate and speed
- detect collisions between player and enemy car
- ends the game when a collision occurs
- tracks and saves user scores and login information using file input/output

How to run the program:
- have java installed
- download the repository
- open the project in an IDE such as VSCode
- run Main.java as main file

Purpose:
- Java swing gui knowledge building
- Event handling with mouse, keyboard, buttons
- OOP design using inheritance and abstract classes
- Collision detection
- Animation with a swing timer
- File input and output for user data and scores
